# Logo provider

Taruh logo provider di folder ini. Akan tampil otomatis di kartu/daftar VPS;
kalau file belum ada, otomatis fallback ke inisial gradient.

## Aturan

- **Format:** `.svg` (ideal) atau `.png` transparan ~128px.
- **Nama file = key provider** (huruf kecil, persis seperti di bawah).
- Diakses di URL `/logos/<key>.svg`.
